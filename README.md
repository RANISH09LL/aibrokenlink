# 🔗 AI Broken Link Fixer Dashboard

A modern, responsive React dashboard that simulates an advanced broken link scanning and AI-powered recovery tool. Built with React, Vite, and Recharts.

## ✨ Features

- **Link Scanning Simulation**: Simulates crawling a website for broken links with a progress bar and live log output.
- **AI Recovery Suggestions**: Mocks an AI service analyzing 404 pages and suggesting redirects or Wayback Machine recovery.
- **Data Visualization**: Includes an Analytics tab with charts built using `recharts` to show scan history and fix rates.
- **Beautiful UI**: Custom styling using a dark-mode first design system, micro-interactions, and modal popups.
- **Responsive Design**: Adapts beautifully to mobile, tablet, and desktop screens.

## 🛠️ Tech Stack

- **React 18**
- **Vite**
- **Recharts** (for data visualization)
- **Vanilla CSS / Inline Styles**

## 🚀 Getting Started

To run this project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/RANISH09LL/aibrokenlink.git
   cd aibrokenlink
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. Open your browser and visit `http://localhost:5173`

## 📦 Deployment

This project is configured to be easily deployed to GitHub Pages.
Run `npm run deploy` to build and deploy the app to the `gh-pages` branch.
